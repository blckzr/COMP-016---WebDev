# Web Development Activity Progress 🚀

This repository serves as a live documentation of my learning journey and activity requirements for my **Web Development** course. The project evolves through three distinct phases of development, showcasing the transition from structure to style, and finally, functionality.

---

## 📌 Project Overview

The goal of this repository is to demonstrate a progressive mastery of front-end technologies. Each milestone marks a significant upgrade in the site's capabilities and user experience.

---

## 🗺️ Roadmap & Progress

### 🧱 Phase 1: Skeleton (HTML Only)

- [x] Semantic HTML5 structure.
- [x] Basic content layout (Headings, Paragraphs, Lists).
- [x] Media integration (Images and Links).
- [x] Forms and Tables for data structure.

### 🎨 Phase 2: Presentation (HTML + CSS)

- [ ] Internal/External CSS implementation.
- [ ] **Flexbox/Grid** for responsive layouts.
- [ ] Custom typography and color palettes.
- [ ] Hover effects and transitions.

### ⚡ Phase 3: Interactivity (HTML + CSS + JS)

- [ ] DOM Manipulation.
- [ ] Event listeners (button clicks, form validation).
- [ ] Dynamic content rendering.
- [ ] API integration or LocalStorage (optional).
